Bikechain

Descripción del proyecto

Bikechain es una aplicación web creada para aplicar y consolidar mis conocimientos en desarrollo blockchain, permitiendo registrar entrenamientos de ciclismo directamente en la red de Ethereum. Con el primer registro, el usuario recibe un NFT único bajo el estándar ERC-721.

El proyecto incluye smart contracts en Solidity, una interfaz en React y la integración con Metamask a través de Ethers.js para facilitar la gestión segura de transacciones. Se han implementado test unitarios para verificar el correcto funcionamiento de la web.

Características destacadas

  • Smart contracts desarrollados en Solidity para la gestión de entrenamientos y creación de NFTs bajo el estándar ERC-721
  • Integración con Metamask mediante Ethers.js para una experiencia segura y fluida
  • Interfaz responsiva creada en React, con enfoque en usabilidad y diseño moderno
  • Visualización de métricas como distancia, tiempo y velocidad promedio
